## 📖 О проекте

**MiniShop Bot** — это Telegram-бот для продажи снеков, напитков и других товаров в **гостиницах, общежитиях и небольших магазинах**. Постояльцы могут заказывать еду в номер, а администратор — управлять ассортиментом и получать уведомления о заказах.

### ✨ Основные возможности

| 👤 Для пользователей | 🛠️ Для администратора |
|---------------------|----------------------|
| 🌐 3 языка (Русский, Узбекский, Английский) | 🔐 Вход по `/admin` |
| 🏪 Каталог с пагинацией | ➕ Добавление товаров |
| 🛒 Корзина с изменением количества | 🗑️ Удаление товаров |
| 📞 Оформление заказа (телефон + комната) | 📦 Уведомления о новых заказах |
| 💾 Автосохранение истории | 📊 База данных SQLite |
